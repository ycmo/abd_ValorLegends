* **測試任務**：

  2. **戰役關卡** (`campaign`)

* **有無消耗高價值資源風險**：**無**。所有的點擊均為功能進入、位置關卡挑戰，未觸發任何鑽石、召喚券、金幣或次數消耗。

---

## 2. 戰場狀態機重要發現：雙擊關閉 (Double-Tap Dismissal)

在試跑過程中，我們發現戰鬥結束後，遊戲會依序彈出兩個不同的結算畫面：
1. **第一層：勝負判定畫面** (Victory/Defeat logo) - 需要點擊任意區域以繼續。
2. **第二層：數據/獎勵結算畫面** (Battle Stats/Rewards overview) - 顯示傷害統計、獲得獎勵等，再次需要點擊任意區域。

這意味著：
* **戰鬥結束後必須執行兩次連續點擊 `(800, 450)`**，每次間隔 3.5 秒左右，才能完全 dismiss 結算畫面並返回原功能頁或主畫面。
* 單次點擊或單純發送 Back 鍵會使畫面卡在第二層結算頁面，導致後續導航比對失敗。此邏輯已完整整合進 `run-small-tasks` 核心流程中。

## 3. 試跑詳細步驟與座標

### 任務 A：戰役關卡 (`campaign`)
1. **大廳進入挑戰**：在掛機大廳點擊挑戰按鈕 `(1387, 712)` 進入戰役準備畫面。
2. **啟動戰鬥**：在準備畫面點擊右下角「戰鬥」按鈕 `(1167, 790)`，戰鬥開始。
3. **戰鬥等待**：等待 25.0 秒，戰鬥順利完成。
4. **結算雙擊關閉**：
   * 點擊 `(800, 450)` 關閉 Victory/Defeat LOGO。
   * 點擊 `(800, 450)` 關閉傷害數據與獎勵。
5. **返回確認**：成功退回掛機大廳頁面（任務按鈕 `task_button.png` 匹配度 `0.9999`）。

### 任務 B：無盡試煉 (`endless_trial`)
1. **大廳切換野外**：在底部選單點擊「野外」分頁 `(477, 840)`。
2. **進入無盡試煉城堡**：在野外地圖中，點擊正中間「無盡試煉」城堡標籤 `(790, 535)` 進入傳送門選擇頁面。
3. **進入特定試煉**：點擊左側啟動中的「無盡試煉」傳送門 `(200, 450)`。
4. **開啟挑戰詳情**：在關卡資訊頁點擊右下角「挑戰」按鈕 `(1500, 825)`。
5. **啟動戰鬥**：在佈陣準備畫面中，再次點擊右下角「挑戰」 `(1500, 825)`，戰鬥開始。
6. **戰鬥等待**：等待 25~30 秒，直至戰鬥結束。
7. **返回流程（自動進入下一關詳情）**：
   * 戰鬥結束後遊戲會自動加載下一關的詳情畫面。
   * 點擊左上角黃色返回鍵 `(50, 60)` 退回試煉傳送門選擇頁。
   * 再次點擊左上角黃色返回鍵 `(50, 60)` 退回野外地圖。此時已回到主畫面（五大分頁之一的野外分頁），任務順利結束。
