# 需求 QA

> 維護方式：Codex 追加問題，使用者填寫答案。
> 最後更新：2026-06-05

這份檔案用來記錄會影響自動化行為的需求問題。之後 Codex 有新問題時，應該追加到這裡，不要只留在聊天紀錄。使用者可以直接編輯每題的「答案」欄位。

## 待回答問題

### Q001. 召喚頁左上箭頭行為

**背景：** 高級召喚頁左上有遊戲內返回箭頭。使用者已補充：左上箭頭可以退出召喚回到主畫面。

**問題：** 點擊高級召喚頁左上箭頭後，是否永遠直接回主畫面？還是有時會出現確認框？如果會出現確認框，自動化應該點哪個按鈕？

**答案：** 左上的箭頭是"上一頁"，如果使用者是從每日任務過去，就會回到每日任務(腳本大多數是如此)。但是為了測試，可能得從大廳進去，就會回到大廳。
這個上一頁，不會有確認框


### Q002. 獎勵彈層關閉方式

**背景：** 免費召喚後可能出現「獲得道具」這類獎勵彈層。

**問題：** 所有任務的獎勵彈層是否都可以點畫面空白處關閉？還是有些彈層必須點特定的確認或關閉按鈕？

**答案：** 通常就都是點比較外圍的空白處關閉。如果你試兩次關不了就截圖用小畫家開啟，我會圈給你。


### Q003. 標準回每日任務流程

**背景：** 一鍵執行需要每條 route 最後都回到每日任務頁。

**問題：** 標準完成流程是否可以設計成：任務頁返回或退出 -> 主畫面 -> 點右上「任務」-> 回到每日任務？如果有例外 route，請列出。

**答案：** 很多時候功能場的返回，就是上一頁的意思(但不一定)，所以如果點了回到每日任務，就沒事，但有的點了會回到主畫面，那就點右上的任務。
另外，主畫面很複雜，有五個分頁，不是任圖形中間，要認底下的分頁條(主城|野外|掛機|英雄|神器)


### Q004. 下一步實作優先順序

**背景：** 目前框架已能進每日任務並 probe 任務列；免費召喚已實機測過，但安全回每日任務流程還需要補齊。

**問題：** 是否同意 Codex 依序實作：回每日任務框架 -> 完整召喚閉環 -> 秘境副本或公會副本？

**答案：** 對，每個任務的終點都回到每日任務。先做秘境副本，比較單純。


### Q005. 已完成或可領取規則

**背景：** 目前 finder 把「找到任務 label，但同列找不到『前往』」視為 `done_or_claimable`，並跳過該任務。

**問題：** 這個規則是否先維持？也就是同列顯示「完成」、「已完成」或「領取」而不是「前往」時，都先跳過不處理。

**答案：** 沒錯。完成有兩個狀態要看右邊按鈕: 灰色已完成 或是 金色領取。
未完成則是藍色前往。
 

### Q006. 秘境副本每日任務完成條件

**背景：** 下一個優先實作目標是秘境副本。每日任務文字是「挑戰/掃蕩任意秘境副本3次」。

**問題：** 秘境副本這條每日任務是否只要完成 3 次掃蕩/挑戰即可？如果已經有足夠次數，是否不需要購買次數？還是固定流程永遠是先購買迷失森林兩次，再掃蕩全部？

**答案：** 其實按全部掃蕩，我忘了事是掃蕩5次或10次，所以一定會超過。重點是迷失森林的前兩次值得購買。所以固定買完以後按全部掃盪。


### Q007. 秘境副本購買次數安全規則

**背景：** 目前理解是「只購買迷失森林兩次，按 + 一次，再購買」。

**問題：** 如果進入秘境副本後看到購買次數不是預期畫面，例如已經買過、鑽石不足、價格不同、或不是迷失森林，是否應立即停下截圖，不繼續點？

**答案：** 對，只要不確定就停下來截圖，用小畫家開啟


### Q008. 秘境副本掃蕩後彈層處理

**背景：** 掃蕩後可能出現獎勵彈層或確認彈層。你已說通常點外圍空白處可關閉，兩次不行就截圖開小畫家。

**問題：** 秘境副本掃蕩後是否也套用這個通用規則：先點外圍空白處最多兩次，仍關不掉就停止、截圖、開小畫家？

**答案：** 對，不確定就是停止、提問、截圖開小畫家。 這是萬用作法。


### Q009. 秘境副本返回路徑

**背景：** route 終點必須回每日任務。你已說很多返回鍵是「上一頁」，可能回每日任務，也可能回主畫面。

**問題：** 秘境副本完成後，是否優先點左上返回箭頭？如果返回到秘境選單/主畫面/每日任務，分別應該怎麼走？是否有任何會跳確認框的退出按鈕要避開？

**答案：** 對優先按左上角(上一頁)返回，回主畫面就多按一次右上角任務。應該不會有確認框，發生不預期或不確認情況就是萬用做法。


### Q010. 點金手 20/50 鑽使用次數

**背景：** 既有答案寫「點金手要全點完，免費+20+50」。舊實機截圖顯示 20 鑽與 50 鑽可能各有多次剩餘次數，例如 20 鑽剩 3 次、50 鑽剩 3 次。

**問題：** 點金手自動化應該是「免費點一次、20 鑽點一次、50 鑽點一次」就停止，還是要把允許的免費/20/50 三個等級剩餘次數全部用完，直到三個按鈕都變灰？

**答案：** 允許全部都點完
