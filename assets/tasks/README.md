# Task Templates

每個任務放在自己的資料夾：

```text
assets/tasks/arena/
assets/tasks/bounty/
assets/tasks/campaign/
assets/tasks/endless_trial/
assets/tasks/guild_dungeon/
assets/tasks/guild_wish/
assets/tasks/midas/
assets/tasks/secret_realm/
assets/tasks/summon/
assets/tasks/time_travel/
```

每個任務至少需要 `task_label.png`，用來在每日任務列表辨識該任務列。
