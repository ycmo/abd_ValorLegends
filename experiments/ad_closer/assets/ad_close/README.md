# Ad Close Templates

廣告關閉按鈕的 template 圖片目錄。

## 命名建議
- `close_x_01.png`   → 廣告右上角「X」按鈕
- `close_x_02.png`   → 另一款廣告的「X」樣式
- `skip_01.png`      → 「Skip Ad」跳過按鈕
- `close_btn_01.png` → 「CLOSE」文字按鈕

## 裁切方式
1. 執行 `python src/main.py screenshot` 擷圖
2. 用圖片軟體裁切 close/X/skip 按鈕本身（約 40~100px）
3. 存為 PNG 放入此目錄

詳見 `experiments/ad_closer/README.md` 第 3 節。
