# 異界奇聞：全自動導航開發報告

## 1. 任務目標
目前的任務是讓腳本能夠**自動判斷當前畫面**，並且無論在哪個分頁（主城、野外、掛機），都能自動導航進入「異界奇聞」去執行點擊廣告的任務。

進入的邏輯路徑為：
1. 點擊左側的「王國事件」圖示。
2. 點擊展開選單中的「異界奇聞」長條按鈕。

## 3. 目前遭遇的技術瓶頸 (Blocker)

雖然程式能夠**100% 成功辨識**上述兩個特徵在畫面上的精確位置，但卡在第一步的「點擊」動作上。

**詳細症狀：**
1. 程式透過底層 ADB 送出點擊指令（包含單擊、雙擊、長按 100ms）至 `(45, 340)` 或 `(45, 370)` 座標。
2. 相同的點擊指令，若是點擊下方的「主城」或「英雄」標籤，皆可瞬間完成切換。
3. **唯獨點擊左側的「王國事件」圖示，遊戲畫面無任何反應，選單未能順利展開。**

> [!WARNING]
> 
> 這可能與特定 UI 框架阻擋模擬點擊有關，或者該按鈕在某些遊戲狀態下（例如正在播放特效、被其他隱形 UI 遮擋）無法透過點點座標的方式觸發。

## 4. 下一步行動計畫
在不修改程式的前提下，目前已經將需求與截圖紀錄完成。
後續若要克服這個點擊失效的問題，我們可能需要測試以下方案：
- **微調座標點擊點**：嘗試點擊圖示的其他邊緣（例如偏左或偏右）。
- **觸控事件注入 (Sendevent)**：改用更底層的 Linux input device 事件來模擬人類手指的按壓與抬起，繞過遊戲引擎對單純 `input tap` 的防禦機制。
